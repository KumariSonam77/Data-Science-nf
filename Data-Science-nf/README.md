# Employment of India Dataset - Data Cleaning Project

## Project Overview

This project focuses on cleaning the Messy Employment India Dataset using Python and Pandas.

## Data Cleaning Steps Performed

1. Load and inspect the dataset
2. Understand dataset structure and data types
3. Handle missing values
4. Remove duplicate records
5. Correct inconsistent formatting and text values
6. Fix data type issues
7. Detect and handle outliers
8. Rename and standardize column names
9. Validate cleaned data
10. Export the final cleaned dataset

* Python
* Pandas
* Google Colab
